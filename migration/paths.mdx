<Info>
  **Classical** paths fix today's compliance and hygiene gaps. **Quantum** paths prepare your infrastructure for post-quantum cryptography before 2030. **Hybrid** paths deploy both algorithms simultaneously so no client breaks during the transition.
</Info>

## Reading the table

- **Template ID** is the identifier passed to the `POST /api/v1/migration/plans` endpoint when creating a plan programmatically.
- **Monitoring hours** is the minimum window after canary deployment before full rollout is authorized. Hybrid paths have longer windows because two certificate chains must be validated concurrently.
- Platform abbreviations: ACM = AWS Certificate Manager, AKV = Azure Key Vault, GCP CM = Google Certificate Manager, ALB = AWS Application Load Balancer.
